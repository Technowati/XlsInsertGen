CsvInsertGen - Generate SQL Inserts From CSV File

Often uploading a CSV file to SQL server via import takes more effert than you'd intitally expect. Taking a diffrent approach, this utility converts the data into a series of SQL Inserts that can be run like any other script.


Some notes and assumptions

  • All columns are treated as character
  • Column sizes are calculated form reading the whole CSV file
  • By deafult calcualted column size is increased by 10% to allo for some basic manipulation withing the table
  • Commits are done every 1,000 inserts
